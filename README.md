Krushak is a **full-stack AI-driven application** designed to empower farmers with intelligent fertilizer recommendations. It leverages **machine learning models** and **real-time weather data** to provide precise, actionable insights for soil and crop management.

- **Instant Predictions:** Real-time fertilizer suggestions based on soil and weather conditions
